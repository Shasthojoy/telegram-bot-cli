# telegram-bot-cli

Unofficial telegram bot cli

Work In Progress

## Install

```
$ npm install -g telegram-bot-cli
```

## Usage

```
$ telegram-bot "<file_mask>"
$ telegram-bot <chat_id> "<file_mask>"
$ telegram-bot <chat_id> "<file_mask>" --token=<token>
```
